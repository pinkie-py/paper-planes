# Paper Planes

### Getting started

Install Node v24.13 directly or with a version manager like [nvm](https://github.com/nvm-sh/nvm).

Install pnpm

```sh
npm i -g pnpm
```

Install dependencies

```sh
pnpm i
```

Then go into your directory and have fun.

```sh
pnpm run start:dev
```

To run everything at once, run the above at the root of the repo.

Familiarise yourselves with documentation on [Confluence]() and ensure to apply the WoW. If there isn't a ticket already created for the work you want to do, create one yourself. Code without a ticket can't be approved.
